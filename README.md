# Ixolink Landing Page

A professional, standalone landing page for Ixolink - a data-driven lead generation agency.

## Overview

This landing page showcases Ixolink's services in B2B and B2C lead generation, web data solutions, and data verification. Built with modern web technologies and featuring smooth animations powered by GSAP.

## Features

- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Modern Animations**: GSAP-powered scroll animations and transitions
- **Professional Branding**: Ixolink's violet/indigo color scheme with Sora font
- **Interactive Elements**: 
  - Sticky navigation with scroll effects
  - Testimonial carousel with auto-rotation
  - Smooth scroll navigation
  - Contact form with validation
  - Mobile-friendly hamburger menu

## Sections

1. **Hero Section**: Eye-catching introduction with gradient background
2. **About Section**: Core values and company introduction
3. **Mission & Vision**: Company goals and aspirations
4. **Services**: B2B/B2C lead generation and web data solutions
5. **Testimonials**: Client success stories with carousel
6. **Contact**: Contact form and company information
7. **Footer**: Links, social media, and additional information

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS custom properties
- **JavaScript (ES6+)**: Interactive functionality
- **GSAP 3.12**: Advanced animations and ScrollTrigger
- **Google Fonts**: Sora typeface

## Usage

### Local Development

1. Open `index.html` in a web browser
2. No build process required - works directly in the browser

### Deployment

Simply upload all files to your web server maintaining the folder structure:

```
ixolink-landing/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── images/ (add your images here)
└── README.md
```

## Customization

### Colors

Edit CSS custom properties in `css/styles.css`:

```css
:root {
    --color-primary: #7340FF;
    --color-secondary: #1C1751;
    --color-tertiary: #140F23;
    /* ... */
}
```

### Content

Edit text content directly in `index.html` for:
- Company information
- Services descriptions
- Testimonials
- Contact details

### Images

Add your images to the `images/` folder and update image paths in `index.html`.
